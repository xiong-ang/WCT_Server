# WCT_Server
> A back-end framework for web compareTool project.

## How to Run:
1. Install MongoDB
2. Create DB:   
webct   
|  
-----_users  
|  
-----_compareResults  
3. Restor library by nuget
4. Ctrl + F5 on VS
